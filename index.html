Data source & settings
Default source is the open openfootball World Cup 2026 dataset (served from GitHub with CORS, so it works straight from a local file — no server or API key). It updates as results are recorded. You can point this at the 2022 or 2018 dataset to see a completed tournament, or at your own fork. Last good data is cached in your browser so the page still works offline.
